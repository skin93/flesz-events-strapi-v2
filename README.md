# flesz-events-strapi-v2
[FleszEvents](https://fleszevents.pl/) - music portal. Informations about concerts and festivals (mainly in Poland).

## Technologies
- [React](https://reactjs.org/)
- [Next.js](https://nextjs.org/)
- [GraphQL](https://graphql.org/)
- [Strapi CMS](https://strapi.io/)
- [Material-UI](https://mui.com/)
- [Mapbox](https://mui.com/)
